# 📊 FX Reinforcement Learning System（簡易版）


ようこそ！  
このパッケージは、**強化学習技術を応用したFX自動分析・戦略判断システムの学習・実験用テンプレート**です。  
特に「トレンドの方向性」に注目し、**複数のテクニカル指標（SMA、MACD、RSI、OBVなど）を入力とするAIエージェント**を訓練する構造となっています。

> ⚠ **ご注意**：このプログラムは学習・実験・検証を目的としており、**実際の売買注文を出す機能は一切実装されていません**。  
> これらを改良して実際の取引環境に接続することは技術的に可能ですが、  
> 本パッケージの提供者は、それによって生じた損失・損害について一切の責任を負いません。**

---

## 🔍 このパッケージでできること

- ✅ MetaTrader5（MT5）からリアルな為替データ（例：GOLD）を取得
- ✅ テクニカル指標を自動計算し、AIモデルの入力に変換
- ✅ PPO（Proximal Policy Optimization）モデルによる学習・推論
- ✅ 学習済みモデルの保存／再利用による継続学習
- ✅ 行動ログや報酬などの評価結果を可視化

> 💡 実売買システムとして使いたい場合は、ご自身でエントリー／決済ロジック、MT5連携（MQL5やPython-Bridge）などを改良・追加してください。

## 🧠 各モデルの実行手順（例：トレンドAIモデル）

1. **データ準備**  
   MetaTrader5（MT5）からGOLDの価格データ（M5足）を取得

2. **学習環境の構築**  
   `TrendEnv` を用いた強化学習環境をセットアップ

3. **モデルの学習・保存**  
   モデルがあればロード、なければ新規作成して学習

4. **評価**  
   報酬や行動ログを使って学習の成果を可視化



📦 モデル管理(学習フェーズ)について

モデルを継続学習したい場合 → force_train ＝ Falseにしてください。(デフォルトでFalseです。)

モデルを最初から学習し直したい場合 → force_train = True に変更してください。

## 🔧 必要なPythonパッケージ

このシステムを動かすために、以下のPythonライブラリが必要です。

| ライブラリ名         | 用途                                         |
|----------------------|----------------------------------------------|
| `MetaTrader5`        | MT5から為替データを取得                      |
| `pandas`             | データフレームの操作                         |
| `numpy`              | 数値計算全般                                 |
| `matplotlib`         | チャート・グラフの可視化                     |
| `scikit-learn`       | スケーリング処理（StandardScaler）           |
| `gymnasium`          | 強化学習環境の構築                           |
| `stable-baselines3`  | PPOモデルによる強化学習                      |
| `torch`              | ニューラルネットワークの計算（PyTorch）      |
| `ta-lib`　　　　　　 | テクニカル指標の計算（SMA、MACDなど）        |
| `zipfile`（標準）    | モデルファイルの解凍と確認（標準ライブラリ） |
| `collections`（標準）| 行動分布の集計（`Counter`使用）              |
| `os`（標準）         | ファイルパス処理                             |

❓ よくある質問(まだないので想定質問)（FAQ）

Q1. MetaTrader5がインストールされていないと動作しませんか？
→ はい、MetaTrader5 モジュールはインストールが必要です。

Q2. データは自分の通貨ペアで使えますか？
→ はい、symbol = "GOLD" の部分を "USDJPY" などに変更してMT5から取得できます。

Q3. 学習済みモデルを渡された状態から再学習は可能ですか？
→ 可能です。ノートブック内で自動的にモデルをロードし、learn() を続行します。
→上記の”モデル管理(学習フェーズ)について”をご確認ください。
